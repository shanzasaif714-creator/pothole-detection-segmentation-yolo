# Pothole Detection and Segmentation using YOLOv11

A computer vision model that automatically detects and segments road potholes 
using YOLOv11 segmentation architecture. Can assist in smart city infrastructure 
monitoring, road maintenance, and autonomous vehicle safety.

## Results
- ✅ Real-time pothole detection and segmentation
- ✅ Accurately segmented potholes on road surfaces
- ✅ Evaluated with normalized confusion matrix and test results
